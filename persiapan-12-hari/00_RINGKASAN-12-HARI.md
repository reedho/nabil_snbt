## STRATEGI KUNCI — CONTEKAN CEPAT

### Strategi Umum (WAJIB INGAT)
```
1. Two-Pass: mudah dulu, sulit belakangan
2. JANGAN kosongkan jawaban (IRT tidak ada penalti)
3. Stuck > 90 detik → tandai, lanjut
4. 5 menit terakhir → isi semua yang kosong
```

### Strategi B. Inggris
```
1. Baca PERTANYAAN dulu, baru passage
2. Vocabulary → SUBSTITUSI ke kalimat
3. Inference → pilih yang MODERAT (hindari always/never/all/none)
4. Main Idea → harus mencakup SELURUH passage
5. Transition words = petunjuk arah argumen
```

### Strategi IRT
```
- Soal mudah benar = fondasi skor (jangan ceroboh!)
- Soal sulit benar = bonus besar
- Pola konsisten dihargai
- Tebakan acak di soal mudah = berbahaya
- Tebakan tereliminasi di soal sulit = layak dicoba
```
