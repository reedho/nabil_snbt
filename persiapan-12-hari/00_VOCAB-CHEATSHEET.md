| **eventually** | pada akhirnya | secara eventual |
| **address** (v) | menangani | alamat |

## B2. Konjungsi Berpasangan (WAJIB hafal)

| ✓ BENAR | ✗ Salah |
|---|---|
| tidak hanya... **tetapi juga** | tidak hanya... melainkan |
| bukan... **melainkan** | bukan... tetapi |
| baik... **maupun** | baik... ataupun |
| antara... **dan** | antara... dengan |
| jangankan... **, ...pun** | jangankan... bahkan |

## B3. Aturan Tanda Koma (5 aturan emas)

1. **SETELAH** konjungsi penghubung antarkalimat di awal:
   `Selain itu,` · `Oleh karena itu,` · `Namun,` · `Dengan demikian,` · `Sementara itu,` · `Akan tetapi,`
2. **SEBELUM** konjungsi koordinatif: `, tetapi` · `, melainkan` · `, sedangkan`
3. **SETELAH** anak kalimat di awal: "Jika hujan turun**,** kita di rumah."
4. **SEBELUM** nama orang yang disapa: "Terima kasih**,** Pak."
5. **MEMISAHKAN** rincian: "saya**,** kamu**,** dan dia"

## B4. Huruf Kapital (sering salah)

- **Jabatan + nama orang** = kapital: **Gubernur** Andra Soni · **Presiden** Joko Widodo · **Menteri** Pendidikan
- Tanpa nama = kecil: "seorang gubernur datang"
- Nama institusi: **RSUD Dr. Soetomo**
- Nama bulan, hari, tahun baru: **April**, **Senin**, **Tahun Baru**
- Geografi + nama diri: **Kota** Solo, **Provinsi** Banten

## B5. Kata Mubazir (HAPUS)

| Tulisan biasa | Yang efektif |
|---|---|
| saling bantu-membantu | saling membantu / bantu-membantu |
| agar supaya | agar / supaya |

| Kata | Sinonim |
|---|---|
| **kontinu** | terus-menerus (BUKAN konsisten) |
| **impulsif** | tergesa-gesa (BUKAN ceroboh) |

## C3. Analogi Pasangan Kata (rumus hubungan)

> **Identifikasi HUBUNGAN dulu**, baru cocokkan ke opsi.

| Hubungan | Contoh |
|---|---|
| Penghasil → Hasil | guru : pembelajaran = listrik : **penerangan** |
| Sebab → Akibat | virus : penyakit = api : **kebakaran** |
| Bagian → Keseluruhan | bab : buku = pasal : **undang-undang** |
| Alat → Fungsi | pulpen : menulis = pisau : **memotong** |
| Profesi → Tempat | dokter : rumah sakit = guru : **sekolah** |
| Umum → Khusus | hewan : kucing = bunga : **mawar** |

## C4. Denotatif vs Konotatif (cara bedakan)

- **Denotatif** = makna kamus / literal
- **Konotatif** = makna kiasan

**Tes cepat:** bisakah kata ini diganti dengan definisi kamus tanpa kehilangan makna kalimat?
- "Pisaunya tajam" → bisa → **denotatif**
- "Lidahnya tajam" → tidak bisa → **konotatif**

**Contoh dari TO10 (Q17):** "Dipelihara" = literal (denotatif). "Alat", "jembatan", "sendi" semua metaforis (konotatif).

# 💡 Tips Hafal Cepat (3 hari)

1. **Hafal lawan kata berpasangan** — exacerbate ↔ mitigate, undermine ↔ reinforce. Lebih lengket di otak.
2. **Pakai konteks, bukan definisi kamus** — "precarious employment" lebih ingat daripada "precarious" sendiri.
3. **Fokus seksi A2 (kata penjebak) & B1 (kata baku)** — ROI tertinggi, langsung kena soal.
4. **2× sehari**: 10 menit pagi (baca), 5 menit malam (tutup arti, tebak).
5. **Saat ujian**: kalau ketemu kata asing, **tebak dari konteks** (transition words & kalimat sekitar) — jangan tebak asal.
