## Cara Import ke Anki

1. Buka **Anki Desktop** (download gratis: https://apps.ankiweb.net)
2. Klik **File → Import**
3. Pilih file `.tsv` yang ingin di-import
4. Pengaturan import:
   - **Type:** Basic (Front + Back) — atau Basic (Front + Back + Tags) jika ingin tag otomatis
   - **Deck:** pilih atau buat baru, contoh `IELTS::01-Vocab-AWL`
   - **Field separator:** Tab
   - **Allow HTML in fields:** ✅ (ON) — supaya format bold/italic berfungsi
   - **Field 1 → Front, Field 2 → Back, Field 3 → Tags**
5. Klik **Import**. Ulangi untuk file berikutnya.

> **Tip:** Buat deck parent `IELTS` lalu sub-deck per file. Anki akan otomatis bikin hierarki kalau pakai `::` di nama deck.

## Format Card

Setiap card mengikuti pola:
- **Front:** kata/pertanyaan dalam bahasa Inggris ATAU pertanyaan bahasa Indonesia
- **Back:** definisi singkat (Inggris) + terjemahan (Indonesia) + contoh kalimat
- **Tags:** untuk filter (misal `ielts::vocab::awl`, `ielts::writing::task2`)

Contoh:
```
benefit	(noun) keuntungan/manfaat. Synonyms: advantage, gain, profit. Example: "Regular exercise has many health benefits."	ielts vocab awl
```

---

## Tips Penggunaan

1. **Konsisten > Lama.** 30 menit/hari rutin lebih baik dari 3 jam sekali seminggu.
2. **Jangan skip review.** Spaced repetition rusak kalau ditunda berhari-hari.
3. **Baca contoh kalimat keras-keras** — bantu listening + pronunciation.
4. **Setelah hafal definisi**, coba pakai kata di tulisan/percakapan sehari-hari.
5. **Audio tambahan (opsional):** install add-on Anki "AwesomeTTS" atau "HyperTTS" untuk auto-generate audio dari teks.
